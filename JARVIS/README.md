# JARVIS

HARP 프로젝트에 대한 냉정한 제품 분석과 레퍼런스 리서치를 구조화한 폴더입니다.

## 구성
- `00_overview.md` — 한눈에 보는 총평 / 핵심 판단
- `01_harp_brutal_review.md` — HARP 제품 자체에 대한 냉정한 평가
- `02_commercial_reference_scan.md` — 차용할 만한 상용 SaaS 레퍼런스
- `03_open_source_reference_scan.md` — 차용할 만한 오픈소스 레퍼런스
- `04_enterprise_gap_analysis.md` — 엔터프라이즈 HR 제품 기준 갭 분석
- `05_priority_roadmap.md` — 실제로 제품을 강하게 만들기 위한 우선순위

## 메모
- 목적은 "비슷한 걸 찾기"가 아니라, **그대로 차용해도 될 정도로 실전적인 제품 패턴**을 추출하는 것입니다.
- 상용/오픈소스 모두 포함하며, HARP를 더 강하게 만드는 데 필요한 기능/UX/구조를 중심으로 정리했습니다.
